# 🛡️ Bandit Survival Guide

Bienvenido a mi repositorio de documentación técnica para los retos de **Bandit** de OverTheWire. Este proyecto es parte de mi **Master Plan ISC 2.0**, enfocado en fortalecer mis habilidades en Ciberseguridad Ofensiva y administración de sistemas Linux.

## 🚀 Objetivo
Documentar el uso de comandos, conceptos de redes y técnicas de explotación de vulnerabilidades aprendidas durante los niveles de Bandit.

## 📂 Contenido Principal
* 📖 [**Glosario de Comandos**](./GLOSARIO_BANDIT.md): Un compendio detallado de herramientas de Bash y Linux utilizadas en los retos.

## 🛠️ Tecnologías Usadas
* **OS:** Kali Linux 🐉
* **Tools:** SSH, Bash, Git, Vim/Nano
* **Plataforma:** OverTheWire (Bandit)
